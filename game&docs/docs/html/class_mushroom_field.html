Detailed Description

Contains code for constructing the MushroomField object The MushroomField object holds all the Mushroom Objects. The MushroomField controls the spawning of Mushroom on the field when the game starts. This placement of Mushrooms is random and gets effected by chance that gets altered based on the game difficulty selected.
