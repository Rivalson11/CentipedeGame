Detailed Description

Contains code for constructing the Shoot object The Shoot object is a user controlled creation. The Shoot object moves up until collision occurs. The shoot mechanic has a reload time stopping continuous firing. When the shoot object collides with the wall, Centipede segment, Spider, Flea, Mushroom, or DDT bomb the shoot object gets erased.
