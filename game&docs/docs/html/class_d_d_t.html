Detailed Description

Contains code for constructing DDT bomb object on DDTField vector The DDT bomb randomly spawns on the screen with a limit of 4 bombs at a time. If shot, the DDT bomb explodes. When the DDT bomb explodes, all game objects within a range is destroyed.
